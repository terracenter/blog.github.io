# Host y Grupos de Hots

## ¿Qué es un "host"?
En Zabbix, un "host" se refiere a cualquier dispositivo físico o virtual, aplicación, servicio o cualquier otra colección de parámetros monitorizados relacionados lógicamente.

La creación de hosts es una de las primeras tareas de monitorización en Zabbix. Por ejemplo, si desea supervisar algunos parámetros en un servidor "x", primero debe crear un host llamado, digamos, "Servidor X" y entonces usted puede mirar para agregar elementos de monitoreo a la misma.

Proceda a [crear y configurar un host](7.11.Configurar_un_host.md).
