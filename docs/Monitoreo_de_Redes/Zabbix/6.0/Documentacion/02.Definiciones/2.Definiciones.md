# 2 Definiciones

## Visión general
En esta sección puedes aprender el significado de algunos términos de uso común en Zabbix.

## Definiciones
#### [host](../07.Configuracion/7.1.Hosts_and_host_groups.md)

Cualquier dispositivo físico o virtual, aplicación, servicio o cualquier otra colección de parámetros supervisados relacionados lógicamente.

#### [host group](../07.Configuracion/7.1.Hosts_and_host_groups.md)
Una agrupación lógica de hosts; puede contener hosts y plantillas. Los hosts y plantillas de un grupo de hosts no están vinculados entre sí. Los grupos de hosts se utilizan cuando se asignan derechos de acceso a hosts para diferentes grupos de usuarios.
