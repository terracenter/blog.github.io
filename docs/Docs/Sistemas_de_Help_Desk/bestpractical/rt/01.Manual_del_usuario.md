Historia de este manual
---

En abril de 2003, [BestPractical](https://rt-wiki.bestpractical.com/wiki/BestPractical) publicó un borrador de manual de usuario para RT como PDF descargable. Ese manual se convirtió en un punto de partida para que esta parte del wiki fuera actualizada y ampliada por la comunidad. La documentación para [RT](https://docs.bestpractical.com/rt/latest) y [RTIR](https://docs.bestpractical.com/rtir/latest) también está disponible en el sitio web de [Best Practical](https://bestpractical.com/docs).
